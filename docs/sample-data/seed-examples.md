# Seed Examples

Use these sample payloads to populate the services manually after the stack is running.

## Staff Account

```json
{
  "email": "admin@example.com",
  "full_name": "Store Admin",
  "password": "strongpass123"
}
```

## Customer Account

{
  "email": "customer@example.com",
  "full_name": "Customer One",

}
